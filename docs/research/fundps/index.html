TL;DR

FunDPS introduces guidance in diffusion sampling directly on function spaces, enabling accurate PDE solution inference in as few as 1–2 steps on several benchmarks.

Core method (math)

Template for Diffusion. Paper-specific equations are added when manually curated.

\[x_t = \alpha(t)\,x_0 + \sigma(t)\,\epsilon\ ,\ \epsilon\sim\mathcal{N}(0,I)\] \[\min_\theta\ \mathbb{E}_{t,x_0,\epsilon}\ \|\epsilon-\epsilon_\theta(x_t,t,c)\|^2\ \ \text{(conditioning }c: \text{measurements/masks)}\] \[\text{Sampling: iterate a reverse process so }x_0\sim p_\theta(\cdot\mid c)\ \text{matches observations + physics}\]

Main contribution

  • Formulates diffusion guidance on function spaces (not pixel grids) for PDE operators.
  • Demonstrates strong 1–2 step sampling while keeping errors competitive.
  • Applies to both forward and inverse (partial observation) PDE problems.
