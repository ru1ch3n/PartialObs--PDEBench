TL;DR

DiffusionPDE uses a conditional diffusion model to generate PDE solutions consistent with sparse observations, enabling both forward prediction and inverse reconstruction.

Core method (math)

Template for Diffusion. Paper-specific equations are added when manually curated.

\[x_t = \alpha(t)\,x_0 + \sigma(t)\,\epsilon\ ,\ \epsilon\sim\mathcal{N}(0,I)\] \[\min_\theta\ \mathbb{E}_{t,x_0,\epsilon}\ \|\epsilon-\epsilon_\theta(x_t,t,c)\|^2\ \ \text{(conditioning }c: \text{measurements/masks)}\] \[\text{Sampling: iterate a reverse process so }x_0\sim p_\theta(\cdot\mid c)\ \text{matches observations + physics}\]

Main contribution

  • Introduces diffusion-based PDE solving under partial observation (mask-conditioned generation).
  • Unified framework for forward and inverse problems across multiple PDE families.
  • Shows robustness when measurements are sparse / missing compared to deterministic operators.
